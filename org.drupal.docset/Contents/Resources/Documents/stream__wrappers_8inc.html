Detailed Description

Drupal stream wrapper interface.

Provides a Drupal interface and classes to implement PHP stream wrappers for public, private, and temporary files.

A stream wrapper is an abstraction of a file system that allows Drupal to use the same set of methods to access both local files and remote resources.

Note that PHP 5.2 fopen() only supports URIs of the form "scheme://target" despite the fact that according to RFC 3986 a URI's scheme component delimiter is in general just ":", not "://". Because of this PHP limitation and for consistency Drupal will only accept URIs of form "scheme://target".

See also:
http://www.faqs.org/rfcs/rfc3986.html
http://bugs.php.net/bug.php?id=47070

Enumeration Type Documentation

enum STREAM_WRAPPERS_ALL

Stream wrapper bit flags that are the basis for composite types.

Note that 0x0002 is skipped, because it was the value of a constant that has since been removed. Stream wrapper bit flag -- a filter that matches all wrappers.

enum STREAM_WRAPPERS_HIDDEN

Composite stream wrapper bit flags that are usually used as the types. Stream wrapper type flag -- not visible in the UI or accessible via web, but readable and writable. E.g. the temporary directory for uploads.

enum STREAM_WRAPPERS_LOCAL

Stream wrapper bit flag -- refers to a local file system location.

enum STREAM_WRAPPERS_LOCAL_HIDDEN

Stream wrapper type flag -- hidden, readable and writeable using local files.

enum STREAM_WRAPPERS_LOCAL_NORMAL

Stream wrapper type flag -- visible, readable and writeable using local files.

enum STREAM_WRAPPERS_NORMAL

Stream wrapper type flag -- the default when 'type' is omitted from hook_stream_wrappers(). This does not include STREAM_WRAPPERS_LOCAL, because PHP grants a greater trust level to local files (for example, they can be used in an "include" statement, regardless of the "allow_url_include" setting), so stream wrappers need to explicitly opt-in to this.

enum STREAM_WRAPPERS_READ

Stream wrapper bit flag -- wrapper is readable (almost always true).

enum STREAM_WRAPPERS_READ_VISIBLE

Stream wrapper type flag -- visible and read-only.

enum STREAM_WRAPPERS_VISIBLE

Stream wrapper bit flag -- exposed in the UI and potentially web accessible.

enum STREAM_WRAPPERS_WRITE

Stream wrapper bit flag -- wrapper is writeable.

enum STREAM_WRAPPERS_WRITE_VISIBLE

Stream wrapper type flag -- visible, readable and writeable.
