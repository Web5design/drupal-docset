Detailed Description

Drupal stream wrapper extension.

Extend the StreamWrapperInterface with methods expected by Drupal stream wrapper classes.

Member Function Documentation

DrupalStreamWrapperInterface::chmod ( $  mode)

Changes permissions of the resource.

PHP lacks this functionality and it is not part of the official stream wrapper interface. This is a custom implementation for Drupal.

Parameters:
$modeInteger value for the permissions. Consult PHP chmod() documentation for more information.
Returns:
Returns TRUE on success or FALSE on failure.

DrupalStreamWrapperInterface::getExternalUrl ( )

Returns a web accessible URL for the resource.

This function should return a URL that can be embedded in a web page and accessed from a browser. For example, the external URL of "youtube://xIpLd0WQKCY" might be "http://www.youtube.com/watch?v=xIpLd0WQKCY".

Returns:
Returns a string containing a web accessible URL for the resource.

Implemented in DrupalTemporaryStreamWrapper, DrupalPrivateStreamWrapper, and DrupalPublicStreamWrapper.

DrupalStreamWrapperInterface::realpath ( )

Returns canonical, absolute path of the resource.

Implementation placeholder. PHP's realpath() does not support stream wrappers. We provide this as a default so that individual wrappers may implement their own solutions.

Returns:
Returns a string with absolute pathname on success (implemented by core wrappers), or FALSE on failure or if the registered wrapper does not provide an implementation.

Implemented in DrupalLocalStreamWrapper.
