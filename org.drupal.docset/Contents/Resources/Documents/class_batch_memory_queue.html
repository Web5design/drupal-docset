Detailed Description

Defines a batch queue for non-progressive batches.

Member Function Documentation

BatchMemoryQueue::claimItem ( $  lease_time = 0)

Overrides MemoryQueue::claimItem().

Unlike MemoryQueue::claimItem(), this method provides a default lease time of 0 (no expiration) instead of 30. This allows the item to be claimed repeatedly until it is deleted.

Reimplemented from MemoryQueue.

BatchMemoryQueue::getAllItems ( )

Retrieves all remaining items in the queue.

This is specific to Batch API and is not part of the DrupalQueueInterface.
